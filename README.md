# Google Mobile Ads for Flutter

[![google_mobile_ads](https://github.com/Dimvy-Clothing-brand/googleads-mobile-flutter/actions/workflows/google_mobile_ads.yaml/badge.svg)](https://github.com/Dimvy-Clothing-brand/googleads-mobile-flutter/actions/workflows/google_mobile_ads.yaml)

This repository contains the source code for the Google Mobile Ads Flutter plugin, which enables publishers to monetize [Flutter](https://flutter.dev/) apps using the Google Mobile Ads SDK.

## Documentation

For instructions on how to use the plugin, please refer to the developer guides:
- [AdMob](https://developers.google.com/admob/flutter/quick-start)
- [Ad Manager](https://developers.google.com/ad-manager/mobile-ads-sdk/flutter/quick-start)

## Downloads

Visit [pub.dev](https://pub.dev/packages/google_mobile_ads/versions) for the latest releases of the plugin.

## Getting Started

To get started with Flutter and this plugin:
1. Follow the Flutter setup guide in the [official documentation](https://flutter.dev/docs/get-started/install).
2. Refer to the [example project](https://github.com/Dimvy-Clothing-brand/googleads-mobile-flutter/tree/main/packages/google_mobile_ads/example) in this repository for a demonstration of how to use the plugin.
